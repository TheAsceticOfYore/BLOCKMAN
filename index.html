BLOCKMAN

Network-level Ad Blocking application software

Features

BLOCKMAN is a network-level advertisement blocking application. It uses a host file to block advertisements and malware. A host file is based on Steven Black's host file which blocks fakenews, gambling, porn, social sites. By using the host file, BLOCKMAN provides network-wide protection, block of in-app advertisements and improve of network performance by preventing ads from loading.
